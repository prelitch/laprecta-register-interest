Laprecta

A racing strategy game where you build the track as you play.

What is Laprecta?

Laprecta is a competitive lap‑racing strategy game where players build the track as they move through it. Each turn adds new spaces, triggers Bless or Curse effects, and shifts momentum through forward or backward movement. Checkpoints, traps, and timing windows create constant decisions.

Prototype in development — not final.
