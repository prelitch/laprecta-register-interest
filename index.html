Laprecta

Laprecta is a racing strategy game where you build the track as you play.

What is Laprecta?

Laprecta is a competitive lap‑racing strategy game where players build the track as they move through it. Each turn adds new spaces, triggers Bless or Curse effects, and shifts momentum through forward or backward movement. Checkpoints, traps, and timing windows create constant decisions. It’s built for players who enjoy clear rules, tactical movement, and shared‑board strategy.

Prototype in development — not final, not for sale.

How the Game Works

  1. Players draw and place Space Cards to build the track as they move.
  2. Dice cards push players forward or backward each turn.
  3. Bless and Curse cards create tactical twists and timing decisions.
